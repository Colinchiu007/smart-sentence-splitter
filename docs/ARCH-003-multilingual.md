# ARCH-003 — 多语言详细设计

## 支持的语言
| 语言 | 优先级 | 分词器 | 缩写词表 | 状态 |
|------|--------|--------|---------|------|
| zh (中文) | P0 | jieba | 16 个内置 | ✅ v0.1 |
| en (英文) | P0 | 空白分词 | 35+ 个内置 | ✅ v0.1 |
| mixed (中英混) | P0 | 段落级路由 | 全部 | ✅ v0.1 |
| ja (日文) | P2 | fugashi（待集成）| — | 🚧 |

## 路由逻辑
```python
def detect_language(text):
    cjk_ratio = CJK字符数 / 有效字符数
    ascii_ratio = ASCII字母数 / 有效字符数
    if 假名 > 0.1 → ja
    if CJK > 0.3 → zh (此时 if 英文 > 0.2 → mixed)
    if ASCII > 0.9 → en
    → mixed (兜底)
```
